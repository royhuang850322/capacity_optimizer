# 文档归档说明

本目录用于存放已经不再作为当前交付主文档使用的材料。

归档原则：

- `handoff/`
  历史交接、接力、kickoff、复用提示类文档。
- `planning/`
  已完成阶段的规划、里程碑、安装/打包准备文档。
- `release_notes/`
  旧版本发布说明。

当前仍作为正式入口使用的文档，保留在 `docs/` 根目录，例如：

- `CAPACITY_OPTIMIZER_USER_MANUAL_CN.md`
- `TECHNICAL_REFERENCE_CN.md`
- `developer_guide.md`
- `user_guide.md`
- `desktop_launcher_usage.md`
- `runtime_directory_strategy.md`
- `pyinstaller_onefolder_build.md`
- `CHANGELOG.md`
